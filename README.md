# Text Image Reveal Animation

A smooth scrolling animation that reveals images within text as you scroll down the page, creating an engaging visual experience.

## Description

This project demonstrates a text-image reveal effect using GSAP's ScrollTrigger plugin. As users scroll through the page, images embedded within text lines animate into view, creating a dynamic and interactive storytelling experience.

## Features

- Smooth scrolling with Lenis
- Image reveal animations triggered by scroll position
- Responsive design
- Clean, modern typography using Google Fonts

## Technologies Used

- **HTML5**: Structure and content
- **CSS3**: Styling and layout
- **GSAP (GreenSock Animation Platform)**: Animation library
- **ScrollTrigger**: GSAP plugin for scroll-based animations
- **Lenis**: Smooth scrolling library

## How to Use

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Scroll down to see the image reveal animations

## Browser Support

Works in all modern browsers that support ES6+ and CSS Grid/Flexbox.

## Dependencies

The project uses CDN-hosted libraries:

- GSAP 3.13.0
- Lenis 1.1.13

No installation required - everything is included via CDN links.
